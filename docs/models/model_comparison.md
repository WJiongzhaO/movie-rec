|---|---|---|---|
| Popularity | **1.00** | **0.66** | 热门兜底, 非个性化 |
| ItemCF | 0.00 | 0.00 | 冷用户无协同邻居 |
| SVD | 0.00 | 0.00 | 冷用户无隐因子 |
| ALS | 0.00 | 0.00 | 同 SVD, 无法服务新用户 |
| Content | 0.00 | 0.00 | 冷用户无历史 → 无画像 |

## 模型选型建议

| 场景 | 推荐模型 | 理由 |
|---|---|---|
| "猜你评分" (评分预测) | **SVD** | RMSE 0.890 最优 |
| "猜你想看" (个性化排序) | **ItemCF** | Recall@10 0.081, NDCG@10 0.115 最高 |
| 长尾冷门电影发现 | **Content** | Coverage 80%, Novelty 6.85 |
| 综合体验 (精度+多样性) | **Hybrid** | RMSE 0.922, 兼顾 recall 与 diversity |
| 新用户首屏推荐 | **ColdStart** | 服务率 100%, 偏好问卷 → 即时推荐 |
| 算法对比基线 | **Popularity** | 非个性化下限 |
| 优化方法对比 | **ALS vs SVD** | SGD 在该数据集上显著优于 ALS 闭式解 |

## 关键方法论发现

1. **SVD 粗筛 vs 完整训练的教训**: 网格搜索用 n_epochs=8 省时, 误选 n_factors=100; 完整训练 (n_epochs=20) 复核后真正最优为 n_factors=50。说明**小 epoch 低估大模型**, 最终超参以完整训练为准。

2. **ItemCF top-N 邻居截断的修复**: 调参前发现 `topk_neighbors` 声明但从未使用 (所有取值指标完全一致); 修复了 `itemcf.py` 的邻居截断逻辑后, 该超参才开始生效, 网格搜索定位到 `topk_neighbors=120`。体现"检测方法有缺陷时先修实验环境"的方法论。

3. **Hybrid alpha 曲线可解释**: α=0 (纯内容) 最弱; 升至 α=0.75 (CF 主导) 达峰值; α=1.0 (纯 CF) 反而回落——**内容补充对召回有正向贡献**。

4. **ALS vs SVD**: 在 4.5% 密度下, SGD 的随机梯度天然正则化效果好于 ALS 的闭式解; ALS 更适合隐式反馈 (0/1 交互信号) 场景。
